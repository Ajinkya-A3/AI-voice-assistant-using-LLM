Raven : A binary tree is a tree data structure wherein each node has at most two children, referred to as the left child and the right child. The idea behind a binary tree is to ensure that the tree remains divided into smaller parts, making it easier to search for and retrieve data. Binary trees are commonly used in computer science algorithms, including search algorithms and sorting algorithms.

A binary tree is made up of nodes that contain a data value and two references or pointers to the left child and the right child. The left child node is always smaller than the parent node, while the right child is always larger. This ordering of the nodes makes it easy to search the binary tree since a search can quickly determine whether to go left or right from each node based on the key value being searched. 

The root node of the binary tree is the topmost node, and it has no parent. Each non-root node has one parent node. A leaf node is a node that has no child node, and the height of a binary tree is the number of nodes on the longest path from the root node to a leaf node.

class Node:
